### Install MySQL Connector/J

Agar JSP dapat terkoneksi dengan MySQL, kita membutuhkan **MySQL Connector/J**.

**Langkah-langkah:**

1. Pilih _Platform Independent_ dan download.
2. Ekstrak file `.zip` lalu cari file `mysql-connector-j-x.x.x.jar`.
3. Di project NetBeans kamu:

   - Pilih `Add JAR/Folder`
   - Tambahkan file `.jar` yang telah diekstrak.

## Membaca Data (Read)

Contoh membaca data dari tabel `daftar` dan menampilkannya ke halaman JSP.

```java
<%@ page import="java.sql.*" %>
<%
    String DBDRIVER = "com.mysql.cj.jdbc.Driver";
    String DBCONNECTION = "jdbc:mysql://localhost:3306/test";
    String DBUSER = "root";
    String DBPASS = "";

    Connection conn = null;
    Statement st = null;
    ResultSet rs = null;

    try {
        Class.forName(DBDRIVER);
        conn = DriverManager.getConnection(DBCONNECTION, DBUSER, DBPASS);
        st = conn.createStatement();
        rs = st.executeQuery("SELECT * FROM daftar");
    } catch (Exception e) {
        out.println("Error");
    }
%>
<html>
<body>
    <% while (rs.next()) { %>
        ID: <%= rs.getInt("id") %><br>
        Nama: <%= rs.getString("nama") %><br>
        Nilai: <%= rs.getInt("nilai") %><br>
        <hr>
    <% }
       rs.close(); st.close(); conn.close();
    %>
</body>
</html>
```
