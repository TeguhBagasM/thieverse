# Cara Jalankan Ngrok dengan Laravel (untuk Integrasi Midtrans)

Jika kamu sedang mengembangkan **fitur pembayaran Midtrans** di Laravel, kamu membutuhkan **URL publik** agar Midtrans bisa mengirim **notifikasi callback** ke server lokal. Solusinya adalah **Ngrok**.

Di tutorial ini, kamu akan belajar **cara menjalankan Ngrok** untuk menghubungkan Laravel lokal dengan Midtrans secara cepat dan mudah.

---

Sebelum mulai, pastikan kamu sudah:

- Menjalankan Laravel secara lokal (`php artisan serve`)
- Menginstal Ngrok
- Memiliki akun Midtrans (mode sandbox)

---

## 1. Jalankan Laravel di Lokal

Pastikan Laravel kamu sudah berjalan di port default.

<Image src="/images/blogs/ngrok/serve.png" width="718" height="304" alt="Laravel serve" />

Buka terminal dan jalankan:

```bash
php artisan serve
```

Setelah itu, buka terminal baru:

<Image src="/images/blogs/ngrok/add-terminal.png" width="718" height="304" alt="Tambah terminal" />

Lalu jalankan:

```bash
npm run dev
```

### 2. Jalankan Ngrok

Buka folder proyek Laravel kamu dan jalankan aplikasi Ngrok (klik dua kali file ngrok jika menggunakan GUI).

<Image src="/images/blogs/ngrok/ngrok.png" width="718" height="304" alt="Aplikasi Ngrok" />

Setelah terbuka, Akan muncul tampilan seperti ini

<Image src="/images/blogs/ngrok/jalankan-ngrok.png" width="718" height="304" alt="NGROK" />

jalankan perintah berikut di terminal:

```bash
ngrok http http://localhost:8000
```

Ngrok akan menghasilkan URL publik, seperti:

<Image src="/images/blogs/ngrok/copy-url-ngrok.png" width="718" height="304" alt="NGROK" />

Catat URL tersebut ‚Äî nanti akan digunakan sebagai base URL untuk Midtrans.

Buka URL tersebut di browser dan klik Visit Site:

<Image src="/images/blogs/ngrok/klik-visit.png" width="718" height="304" alt="NGROK" />
