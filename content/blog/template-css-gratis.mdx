# Langkah 2

Pilih desain yang kamu suka lalu bisa klik live demo dulu atau langsung download

<Image src="/images/blogs/download-demo.png" width="718" height="404" alt="download-demo" />

Jika klik live demo tampilannya akan seperti ini:

<Image src="/images/blogs/demo.png" width="718" height="404" alt="demo" />

Jika klik download maka akan langgsung di download dan ada di direktori Downloads seperti ini

<Image src="/images/blogs/download.png" width="718" height="404" alt="download-demo" />

bisa kamu pindahkan ke folder yang kamu inginkan, contohnya ini saya simpan di Documents > portfolio, lalu kamu klik kanan pada rar nya dan klik extract here / ekstrak disini

<Image src="/images/blogs/ekstrak.png" width="718" height="404" alt="ekstrak" />

Jika sudah, kamu bisa klik folder yang sudah di ekstrak dan pilih salah satu file yang ingin dilihat dengan klik 2 kali pada file, misalnya "index.html".
