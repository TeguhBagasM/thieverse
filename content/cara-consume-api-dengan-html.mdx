# Cara Consume API dengan HTML, JavaScript & Tailwind CSS

## Apa itu Consume API?

**Consume API** artinya menggunakan atau "mengonsumsi" data dari API untuk ditampilkan di aplikasi frontend. Bayangkan seperti memesan makanan di aplikasi ojek online - aplikasi tersebut mengambil data (menu, harga, restoran) dari API dan menampilkannya di layar ponsel Anda.

### üîÑ Proses Consume API:

1. **Frontend** mengirim request ke API
2. **API** memproses request dan mengembalikan data (biasanya JSON)
3. **Frontend** menerima data dan menampilkannya ke user

### üì± Contoh dalam Kehidupan Sehari-hari:

- **Instagram**: App mobile consume API untuk menampilkan foto dan post
- **Tokopedia**: Website consume API untuk menampilkan produk dan harga
- **Google Maps**: Aplikasi consume API untuk menampilkan peta dan rute

---

## Prerequisites

- Pemahaman dasar HTML, CSS, JavaScript
- Browser modern (Chrome, Firefox, Safari, Edge)
- Text Editor (VS Code recommended)
- API endpoint yang sudah jalan (kita akan pakai API Laravel dari tutorial sebelumnya)

/**

function showError(message) {
  const errorDiv = document.getElementById("errorMessage");
  const errorText = document.getElementById("errorText");

  errorText.textContent = message;
  errorDiv.classList.remove("hidden");

  // Auto hide setelah 5 detik

/**
 * Fungsi untuk format tanggal menjadi format yang mudah dibaca
 * @param {string} dateString - String tanggal dari API
 * @returns {string} - Tanggal yang sudah diformat
 */
function formatDate(dateString) {
  const date = new Date(dateString);
  const options = {
    year: "numeric",
    month: "long",
    day: "numeric",
    hour: "2-digit",
    minute: "2-digit",
  };
  return date.toLocaleDateString("id-ID", options);
}

/**
 * Fungsi utama untuk memuat posts dari API
 * Fungsi ini akan dipanggil saat halaman pertama kali dimuat
 * dan saat user melakukan pencarian/filter
 */
async function loadPosts(page = 1) {
  try {
    // Tampilkan loading spinner
    showLoading();

    // Ambil nilai dari input pencarian dan filter
    const search = document.getElementById("searchInput").value;
    const published = document.getElementById("publishedFilter").value;

    // Buat URL dengan parameter query
    let url = `${API_BASE_URL}/posts?page=${page}`;

    // Tambahkan parameter pencarian jika ada
    if (search.trim() !== "") {
      url += `&search=${encodeURIComponent(search)}`;
    }

    // Tambahkan parameter filter published jika ada
    if (published !== "") {
      url += `&published=${published}`;
    }

    console.log("üöÄ Mengirim GET request ke:", url);

    // Kirim request ke API menggunakan fetch()
    const response = await fetch(url, {
      method: "GET", // HTTP Method
      headers: {
        "Content-Type": "application/json", // Format data yang kita kirim
        Accept: "application/json", // Format data yang kita terima
      },
    });

    console.log("üì° Response status:", response.status);

    // Cek apakah response berhasil (status 200-299)
    if (!response.ok) {
      throw new Error(`HTTP Error: ${response.status} - ${response.statusText}`);
    }

    // Convert response ke JSON
    const data = await response.json();
    console.log("üì¶ Data yang diterima:", data);

    // Sembunyikan loading spinner
    hideLoading();

    // Tampilkan posts ke halaman
    displayPosts(data.data || []);

    // Tampilkan pagination jika ada
    if (data.pagination) {
      displayPagination(data.pagination);
    }

    // Update current page
    currentPage = page;
  } catch (error) {
    // Tangani error jika terjadi masalah
    console.error("‚ùå Error saat memuat posts:", error);
    hideLoading();
    showError(`Gagal memuat data: ${error.message}`);
  }
}

/**
 * Fungsi untuk menampilkan daftar posts ke dalam HTML
 * @param {Array} posts - Array berisi data posts dari API
 */
function displayPosts(posts) {
  const container = document.getElementById("postsContainer");

  // Jika tidak ada posts
  if (!posts || posts.length === 0) {
    container.innerHTML = `
            <div class="col-span-full text-center py-12">
                <i class="fas fa-inbox text-6xl text-gray-300 mb-4"></i>
                <h3 class="text-xl text-gray-500 mb-2">Tidak ada artikel</h3>
                <p class="text-gray-400">Belum ada artikel yang tersedia atau sesuai dengan pencarian Anda.</p>
            </div>
        `;
    return;
  }

  // Loop untuk membuat HTML setiap post
  const postsHTML = posts
    .map(
      (post) => `
        <div class="bg-white rounded-lg shadow-md hover:shadow-lg transition duration-200">
            <div class="p-6">
                <!-- Status Badge -->
                <div class="flex items-center justify-between mb-3">
                    <span class="inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium ${
                      post.is_published
                        ? "bg-green-100 text-green-800"
                        : "bg-yellow-100 text-yellow-800"
                    }">
                        <i class="fas ${post.is_published ? "fa-check-circle" : "fa-clock"} mr-1"></i>
                        ${post.is_published ? "Published" : "Draft"}
                    </span>
                    
                    <!-- Action Buttons -->
                    <div class="flex gap-2">
                        <button onclick="editPost(${post.id})" 
                                class="text-blue-600 hover:text-blue-800 text-sm">
                            <i class="fas fa-edit"></i>
                        </button>
                        <button onclick="deletePost(${post.id})" 
                                class="text-red-600 hover:text-red-800 text-sm">
                            <i class="fas fa-trash"></i>
                        </button>
                    </div>
                </div>
                
                <!-- Title -->
                <h2 class="text-xl font-bold text-gray-900 mb-2 line-clamp-2">
                    ${post.title}
                </h2>
                
                <!-- Content Preview -->
                <p class="text-gray-600 mb-4 line-clamp-3">
                    ${post.content.substring(0, 150)}${post.content.length > 150 ? "..." : ""}
                </p>
                
                <!-- Meta Info -->
                <div class="flex items-center text-sm text-gray-500">
                    <i class="fas fa-user mr-1"></i>
                    <span class="mr-4">${post.author}</span>
                    
                    <i class="fas fa-calendar mr-1"></i>
                    <span>${formatDate(post.created_at)}</span>
                </div>
            </div>
        </div>
    `
    )
    .join("");

  // Masukkan HTML ke container
  container.innerHTML = postsHTML;
}

/**
 * Fungsi untuk menampilkan pagination
 * @param {Object} paginationData - Data pagination dari API
 */
function displayPagination(paginationData) {
  const container = document.getElementById("pagination");

  if (paginationData.last_page <= 1) {
    container.innerHTML = "";
    return;
  }

  let paginationHTML = '<div class="flex items-center space-x-2">';

  // Previous button
  if (paginationData.current_page > 1) {
    paginationHTML += `
            <button onclick="loadPosts(${paginationData.current_page - 1})" 
                    class="px-3 py-2 bg-white border border-gray-300 rounded-lg hover:bg-gray-50">
                <i class="fas fa-chevron-left"></i>
            </button>
        `;
  }

  // Page numbers
  for (let i = 1; i <= paginationData.last_page; i++) {
    if (i === paginationData.current_page) {
      paginationHTML += `
                <button class="px-3 py-2 bg-blue-600 text-white rounded-lg">
                    ${i}
                </button>
            `;
    } else {
      paginationHTML += `
                <button onclick="loadPosts(${i})" 
                        class="px-3 py-2 bg-white border border-gray-300 rounded-lg hover:bg-gray-50">
                    ${i}
                </button>
            `;
    }
  }

  // Next button
  if (paginationData.current_page < paginationData.last_page) {
    paginationHTML += `
            <button onclick="loadPosts(${paginationData.current_page + 1})" 
                    class="px-3 py-2 bg-white border border-gray-300 rounded-lg hover:bg-gray-50">
                <i class="fas fa-chevron-right"></i>
            </button>
        `;
  }

  paginationHTML += "</div>";
  container.innerHTML = paginationHTML;
}

/**

function showCreateForm() {
  // Reset form
  isEditing = false;
  editingPostId = null;

/**
 * Fungsi untuk submit post (create atau update)
 * @param {Event} event - Event dari form submit
 */
async function submitPost(event) {
  event.preventDefault(); // Mencegah form submit default

  try {
    // Ambil data dari form
    const title = document.getElementById("postTitle").value.trim();
    const author = document.getElementById("postAuthor").value.trim();
    const content = document.getElementById("postContent").value.trim();
    const isPublished = document.getElementById("postPublished").checked;

    // Validasi data
    if (!title || !author || !content) {
      showError("Semua field wajib diisi!");
      return;
    }

    // Siapkan data yang akan dikirim
    const postData = {
      title: title,
      author: author,
      content: content,
      is_published: isPublished,
    };

    console.log("üì§ Data yang akan dikirim:", postData);

    // Tentukan URL dan method berdasarkan mode (create/edit)
    const url = isEditing ? `${API_BASE_URL}/posts/${editingPostId}` : `${API_BASE_URL}/posts`;
    const method = isEditing ? "PUT" : "POST";

    console.log(`üöÄ Mengirim ${method} request ke:`, url);

    // Disable submit button sementara
    const submitBtn = document.getElementById("submitBtn");
    const originalText = submitBtn.innerHTML;
    submitBtn.innerHTML = '<i class="fas fa-spinner fa-spin mr-2"></i>Menyimpan...';
    submitBtn.disabled = true;

    // Kirim request ke API
    const response = await fetch(url, {
      method: method, // POST untuk create, PUT untuk update
      headers: {
        "Content-Type": "application/json", // Format data JSON
        Accept: "application/json",
      },
      body: JSON.stringify(postData), // Convert object ke JSON string
    });

    console.log("üì° Response status:", response.status);

    // Parse response JSON
    const result = await response.json();
    console.log("üì¶ Response data:", result);

    // Enable submit button kembali
    submitBtn.innerHTML = originalText;
    submitBtn.disabled = false;

    // Cek apakah request berhasil
    if (response.ok && result.success) {
      // Berhasil - tutup modal dan reload data
      closeModal();
      showSuccess(result.message || `Artikel berhasil ${isEditing ? "diperbarui" : "dibuat"}!`);
      loadPosts(currentPage); // Reload posts
    } else {
      // Gagal - tampilkan error
      const errorMessage = result.message || `Gagal ${isEditing ? "memperbarui" : "membuat"} artikel`;

      // Jika ada validation errors, tampilkan detail
      if (result.errors) {
        const errorDetails = Object.values(result.errors).flat().join(", ");
        showError(`${errorMessage}: ${errorDetails}`);
      } else {
        showError(errorMessage);
      }
    }
  } catch (error) {
    console.error("‚ùå Error saat submit post:", error);

    // Enable submit button kembali jika error
    const submitBtn = document.getElementById("submitBtn");
    submitBtn.innerHTML = '<i class="fas fa-save mr-2"></i>Simpan';
    submitBtn.disabled = false;

    showError(`Terjadi kesalahan: ${error.message}`);
  }
}

/**
 * Fungsi untuk mengedit post
 * @param {number} postId - ID post yang akan diedit
 */
async function editPost(postId) {
  try {
    console.log("üîç Mengambil detail post ID:", postId);

    // Ambil detail post dari API
    const response = await fetch(`${API_BASE_URL}/posts/${postId}`, {
      method: "GET",
      headers: {
        "Content-Type": "application/json",
        Accept: "application/json",
      },
    });

    if (!response.ok) {
      throw new Error(`Gagal mengambil detail post: ${response.status}`);
    }

    const result = await response.json();
    console.log("üì¶ Detail post:", result);

    if (result.success && result.data) {
      const post = result.data;

      // Set mode editing
      isEditing = true;
      editingPostId = postId;

      // Update modal title
      document.getElementById("modalTitle").textContent = "Edit Artikel";
      document.getElementById("submitBtn").innerHTML = '<i class="fas fa-save mr-2"></i>Perbarui';

      // Isi form dengan data post
      document.getElementById("postTitle").value = post.title;
      document.getElementById("postAuthor").value = post.author;
      document.getElementById("postContent").value = post.content;
      document.getElementById("postPublished").checked = post.is_published;

      // Show modal
      document.getElementById("postModal").classList.remove("hidden");
    } else {
      showError("Gagal mengambil detail artikel");
    }
  } catch (error) {
    console.error("‚ùå Error saat edit post:", error);
    showError(`Gagal memuat data artikel: ${error.message}`);
  }
}

/**
 * Fungsi untuk menghapus post
 * @param {number} postId - ID post yang akan dihapus
 */
async function deletePost(postId) {
  // Konfirmasi sebelum menghapus
  if (!confirm("Yakin ingin menghapus artikel ini? Tindakan ini tidak dapat dibatalkan.")) {
    return;
  }

  try {
    console.log("üóëÔ∏è Menghapus post ID:", postId);

    // Kirim DELETE request ke API
    const response = await fetch(`${API_BASE_URL}/posts/${postId}`, {
      method: "DELETE", // HTTP Method untuk menghapus
      headers: {
        "Content-Type": "application/json",
        Accept: "application/json",
      },
    });

    console.log("üì° Response status:", response.status);

    // Parse response
    const result = await response.json();
    console.log("üì¶ Response data:", result);

    // Cek apakah berhasil
    if (response.ok && result.success) {
      showSuccess(result.message || "Artikel berhasil dihapus!");
      loadPosts(currentPage); // Reload posts
    } else {
      showError(result.message || "Gagal menghapus artikel");
    }
  } catch (error) {
    console.error("‚ùå Error saat delete post:", error);
    showError(`Gagal menghapus artikel: ${error.message}`);
  }
}

## Langkah 3: Penjelasan Kode JavaScript

### üîß **Struktur Kode dan Penjelasan Detail**

#### **1. Konfigurasi API**

```javascript
const API_BASE_URL = "http://localhost:8000/api";
```

- **Fungsi**: Menyimpan base URL API Laravel
- **Tips**: Ganti sesuai dengan server API Anda
- **Production**: Ganti dengan domain production (https://yourdomain.com/api)

#### **2. Fungsi GET - Mengambil Data**

```javascript
// Request ke API
const response = await fetch(url, {
  method: "GET", // HTTP Method
  headers: {
    "Content-Type": "application/json", // Format data
    Accept: "application/json", // Format response
  },
});

// Cek status response
if (!response.ok) {
  throw new Error(`HTTP Error: ${response.status}`);
}

// Convert ke JSON
const data = await response.json();
```

**Penjelasan langkah demi langkah:**

1. **fetch()** - Fungsi modern untuk HTTP request
2. **method: 'GET'** - Mengambil data dari server
3. **headers** - Memberitahu server format data yang kita kirim/terima
4. **response.ok** - Cek apakah status 200-299 (sukses)
5. **response.json()** - Convert response ke object JavaScript

#### **3. Fungsi POST - Mengirim Data Baru**

```javascript
const postData = {
  title: "Judul Artikel",
  content: "Isi artikel...",
  author: "John Doe",
  is_published: true,
};

const response = await fetch(`${API_BASE_URL}/posts`, {
  method: "POST", // HTTP Method untuk create
  headers: {
    "Content-Type": "application/json",
    Accept: "application/json",
  },
  body: JSON.stringify(postData), // Convert object ke JSON string
});
```

**Perbedaan dengan GET:**

- **method: 'POST'** - Untuk membuat data baru
- **body** - Berisi data yang akan dikirim ke server
- **JSON.stringify()** - Convert JavaScript object ke JSON string

#### **4. Fungsi PUT - Update Data**

```javascript
const response = await fetch(`${API_BASE_URL}/posts/${postId}`, {
  method: "PUT", // HTTP Method untuk update
  headers: {
    "Content-Type": "application/json",
    Accept: "application/json",
  },
  body: JSON.stringify(updatedData),
});
```

**Karakteristik PUT:**

- **URL include ID** - `/posts/1` untuk update post dengan ID 1
- **method: 'PUT'** - Untuk update complete data
- **body** - Data lengkap yang akan menggantikan data lama

#### **5. Fungsi DELETE - Hapus Data**

```javascript
const response = await fetch(`${API_BASE_URL}/posts/${postId}`, {
  method: "DELETE", // HTTP Method untuk hapus
  headers: {
    "Content-Type": "application/json",
    Accept: "application/json",
  },
  // Tidak ada body untuk DELETE
});
```

**Karakteristik DELETE:**

- **method: 'DELETE'** - Untuk menghapus data
- **URL include ID** - Spesifik ID mana yang akan dihapus
- **Tidak ada body** - DELETE tidak butuh data tambahan

```
Access to fetch at 'http://localhost:8000/api/posts'
from origin 'file://' has been blocked by CORS policy
```

**Solusi**:

1. Jalankan HTML melalui web server (Live Server di VS Code)
2. Atau setup CORS di Laravel API

#### **Network Error**

```
TypeError: Failed to fetch
```

**Solusi**:

1. Pastikan API server jalan (`php artisan serve`)
2. Cek URL API sudah benar
3. Cek koneksi internet

#### **JSON Parse Error**

```
Unexpected token < in JSON at position 0
```

**Solusi**:

1. API mengembalikan HTML instead of JSON
2. Cek endpoint API di browser langsung
3. Pastikan Accept header correct
