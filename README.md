# Personal Portfolio Website

Ini adalah proyek website personal portfolio saya yang dibangun menggunakan **Next.js**, **TypeScript**, dan **Tailwind CSS**. Website ini menampilkan profil saya sebagai web developer dan juga dilengkapi dengan fitur blog/artikel menggunakan **MDX** serta didukung oleh **Velite** untuk sistem blog statis.

## ✨ Fitur Utama

- 💼 Halaman portfolio pribadi untuk menampilkan informasi tentang saya
- ✍️ Blog/artikel statis menggunakan [Velite](https://github.com/velitejs/velite) dan **MDX**
- 💬 Komentar pada blog dengan [Giscus](https://giscus.app/)
- 🌙 Mode gelap dan terang
- ⚡ Performa tinggi dengan statis generation
- 📱 Responsif di berbagai ukuran layar
- 🎨 Ikon menarik dari [Lucide React](https://lucide.dev/)

## 🧑‍💻 Tech Stack

- **Next.js** – React Framework untuk SSR dan SSG
- **TypeScript** – Bahasa pemrograman dengan static typing
- **Tailwind CSS** – Utility-first CSS framework
- **Velite** – Static blog engine untuk MDX
- **Giscus** – Komentar berbasis GitHub Discussions
- **Lucide React** – Ikon open source berbasis SVG
